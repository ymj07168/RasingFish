# RasingFish
물고기 키우기 게임

## 클래스 소개
Main - 게임 실행 창  
GameFrame - 게임 실행 창에서 시작 버튼 클릭 시 실행  
Introduction - 게임 실행 창에서 방법 버튼 클릭 시 실행  
Fish - 물고기 쓰레드  
Enemy - 상어(장애물) 쓰레드  
Rule - 게임 방법 설명 탭  
AquaPanel - 수족관 탭  
ButtonPanel - 버튼 관리 탭(돈, 물갈이)  

## 기타 내용
* JAVA 이용  
* 3인 1조로 진행  
* 21/5/7~21/6/14
